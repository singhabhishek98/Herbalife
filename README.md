# Herbalife Manager - React + Ant Design Frontend

## Run karne ka process

```bash
npm install
npm run dev
```

Browser me open karo:

```bash
http://localhost:5173
```

## Build

```bash
npm run build
```

## Features included

- Mobile-first dashboard
- Desktop dashboard with sidebar
- Team members screen
- Members list with search/filter
- Add/Edit member modal
- Renew subscription modal
- Mark Present logic
- Duplicate same-day present check
- Reports screen
- Profile/plans screen
- Mock data included

## Tech stack

- React
- JavaScript
- Ant Design
- Vite
- Dayjs
